# DC Motor Back EMF Investigation 🏎️

This repository contains the experimental data and a Python analysis tool for calculating Back EMF and Motor Efficiency.

### Formula Used:
$$E_b = V - (I_a \times R_a)$$
$$\text{Efficiency} = \frac{E_b \times I_a}{V \times I_a} \times 100\%$$

### Experimental Setup:
- **Voltage Source:** 9V DC
- **Control:** Lab Variable Power Supply
- **Load:** Multi-sized propeller blades
- **Platform:** Breadboard circuit

### Live Tool Features:
- Real-time calculation of Back EMF.
- Instant Efficiency percentage output.
- User-friendly GUI (Tkinter).
